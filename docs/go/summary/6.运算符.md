题目：
```go
// 交换 a 和 b，不允许使用中间变量
var a = 10
var b = 5

// 解决方法 
a = a + b
b = a - b
a = a - b
```

#### 位运算符

十进制转二进制 一直除2取余数就可以了（最后将所有余数反转一下）

& 两个都为1才为1 （100 & 011 = 000）
| 有1就为1 (100 | 011 = 111)
^ 两位不一样为1 （111 ^ 111 = 000 ）
