定时器

time.NewTicker() 返回 ticker 对象 然后通过 for range ticker.C 实现定时器

ticker := time.NewTicker(time.Second * 2)

// for range ticker.C 只会返回 val，没有 key。val 值为当前时间对象
for v := range ticker.C {
    // 每 time.Second * 2 会打印一次
    fmt.Println(v)
    // ticker.Stop() 可以停止定时器
}
