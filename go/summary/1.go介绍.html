安装 go

安装后查看 go version

1.11 版本之后的够不再需要配置 go path，项目也不用放到 go path 下

// 申明包名称
package main;

// 包主函数（入口函数）
func main() {
    
}

go 中没有使用的包都会被移除

运行 go 文件

go run main.go

package main

import "fmt"

func main() {
	fmt.Println("Hello, World!")
}
