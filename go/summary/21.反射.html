# 通过反射获取变量类型

类型种类（Kind）指的是底层类型。

指针类型没有名称（Name），指针种类（Kind）是 ptr

func typeof(x interface{}) reflect.Type {
	return reflect.TypeOf(x)
}

typeof("hehe") // string
typeof(123) // int

# 通过反射获取 空接口 的值

// 通过 reflect.ValueOf() 获取空接口类型的底层类型值
var a interface{} = 1
b := reflect.ValueOf(a) // reflect.Value 1
b.Int() // int 1

reflect.Type 和 reflect.Value 存在的方法 Name Kind Int Float32 Float64

reflect.Int reflect.String
